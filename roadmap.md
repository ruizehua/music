# 产品路线图

> 本文档定义项目的发展路线图，包括版本计划和里程碑。

---

## 版本概览

| 版本 | 状态 | 预计日期 | 重点 |
|------|------|----------|------|
| **v0.1 (MVP) | 🚧 开发中 | 2026 Q2 | 核心播放功能 |
| **v1.0** | ⏳ 计划中 | 2026 Q3 | 完善功能 + 管理后台 |

## v0.1 - MVP 版本 (Minimum Viable Product)

**目标**：可用的个人音乐播放器

**功能清单**：
- [ ] 后端
    - [ ] 项目结构搭建
    - [ ] 音乐上传 API
    - [ ] 音乐列表 API
    - [ ] 音乐流式播放
    - [ ] 歌单 CRUD
    - [ ] 收藏功能
    - [ ] SQLite 数据库设计
- [ ] Android 客户端
    - [ ] 项目结构搭建
    - [ ] 音乐列表页面
    - [ ] 播放页面 (播放/暂停/上一首/下一首)
    - [ ] 歌单页面
    - [ ] 收藏页面
    - [ ] 本地缓存
    - [ ] 后台播放
- [ ] 管理后台 (基础)
    - [ ] 登录
    - [ ] 音乐管理 (列表/上传/删除)

**交付日期**：2026 Q2

## v1.0 - 正式版本

**目标**：功能完整、体验良好的音乐播放器

**功能清单**：
- [ ] 后端
    - [ ] 用户管理 API
    - [ ] 统计 API
    - [ ] 播放历史记录
    - [ ] 配置管理
- [ ] Android 客户端
    - [ ] 歌词显示
    - [ ] 均衡器
    - [ ] 睡眠定时
    - [ ] 播放历史
    - [ ] 自动续播
    - [ ] 桌面小组件
    - [ ] 深色模式切换
- [ ] 管理后台
    - [ ] 完整音乐编辑
    - [ ] 用户管理
    - [ ] 歌单管理
    - [ ] 数据统计仪表盘
    - [ ] 系统配置

**交付日期**：2026 Q3

## v2.0 - 进阶版本

**目标**：锦上添花的功能

**功能清单**：
- [ ] 播放数据统计和展示
- [ ] 进阶音效 (重低音、环绕声)
- [ ] 本地音乐扫描和导入
- [ ] 歌单导入导出
- [ ] 无损音频支持 (FLAC/WAV)
- [ ] 多用户支持
- [ ] 歌词编辑
- [ ] 远程控制
- [ ] 网络电台 (可选)

**交付日期**：2026 Q4
