# 音乐软件需求规范

> 本文件定义了音乐软件的完整功能需求，分为三个层次：核心必选功能、进阶常用功能和锦上添花的可选功能。

---

## 一、核心必选功能（MVP 版本必须做）

这是所有音乐 APP 最基础、用户最高频使用的能力，开发成本极低，技术栈可原生支持。

### 1.1 核心播放控制

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 播放/暂停 | 点击播放或暂停当前歌曲 | ExoPlayer 原生支持 |
| 上一首/下一首 | 切换到上一首或下一首歌曲 | ExoPlayer 原生支持 |
| 进度条拖动 | 支持快进快退，拖动进度条跳转到指定位置 | ExoPlayer 原生支持 |
| 播放模式切换 | 顺序播放、单曲循环、随机播放 | ExoPlayer 原生支持 |

### 1.2 后台与锁屏播放

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 后台播放 | APP 切后台后继续播放音乐 | Android MediaSession |
| 锁屏播放 | 手机锁屏后显示播放控件（切歌、暂停、进度） | Android MediaSession |
| 耳机控制 | 支持插拔耳机自动暂停/播放 | Android 广播监听 |

### 1.3 音频缓存与离线播放

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 自动缓存 | 自动缓存服务端歌曲到手机本地 | ExoPlayer 缓存模块 |
| 离线播放 | 无网络环境也能正常播放已缓存歌曲 | ExoPlayer 缓存模块 |

### 1.4 音乐库管理

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 歌曲列表展示 | 展示所有歌曲列表 | RecyclerView + ViewModel |
| 分类筛选 | 按歌手、专辑、文件名分类筛选 | 数据库查询 + 过滤 |
| 歌曲搜索 | 支持按歌名、歌手搜索 | 数据库模糊查询 |
| 自建歌单 | 添加/移除歌曲、歌单重命名 | Room 数据库 |
| 收藏列表 | 「喜欢/收藏」歌曲单独列表 | Room 数据库 |

### 1.5 基础播放适配

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 音量调节 | 音量同步调节 | Android 音量控制 API |
| 均衡器预设 | 流行、摇滚、古典、原声等预设 | Android Equalizer API |
| 蓝牙设备控制 | 蓝牙耳机/音箱切歌、暂停控制 | MediaSession |

## 二、进阶常用功能（MVP 跑通后迭代）

这部分是基础功能之上的高频体验项，开发成本不高，个人使用实用性极强。

### 2.1 歌词展示

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 逐行滚动歌词 | 播放页逐行滚动显示歌词 | 自定义 View + 歌词解析 |
| 歌词同步 | 歌词与播放进度精准同步 | 时间戳匹配 |
| 桌面悬浮歌词 | 在桌面显示悬浮歌词窗口 | WindowManager |

### 2.2 辅助播放功能

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 睡眠定时 | 10 分钟/30 分钟/1 小时等时长自动停止播放 | AlarmManager |
| 播放历史记录 | 记录播放过的歌曲历史 | Room 数据库 |
| 自动续播 | 启动 APP 自动续播上次暂停的歌曲 | SharedPreferences |

### 2.3 便捷操作能力

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 桌面小组件 | 无需打开 APP 即可快速切歌/暂停 | Android App Widget |
| 全局搜索 | 在系统全局搜索中搜索歌曲 | App Indexing |
| 歌曲信息编辑 | 修改歌名、歌手、专辑名 | Room 数据库更新 |

### 2.4 视觉与个性化

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 专辑封面展示 | 播放页展示专辑封面 | Jetpack Compose 图片组件 |
| 深色/浅色模式 | 支持深色和浅色主题切换 | Jetpack Compose 主题 |
| 自定义主题色 | 用户可自定义主题颜色 | 主题配置 + Compose |
| 播放页动效 | 播放状态变化时的动画效果 | Compose 动画 API |

### 2.5 跨端同步

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 歌单同步 | 歌单数据在多端实时同步 | REST API + 云端数据库 |
| 收藏同步 | 收藏列表跨端同步 | REST API + 云端数据库 |
| 播放进度同步 | 播放进度跨端同步 | REST API + 云端数据库 |
| 播放历史同步 | 播放历史跨端同步 | REST API + 云端数据库 |

## 三、锦上添花的可选功能（按需添加）

这部分属于个性化体验项，个人使用可根据需求选择性开发。

### 3.1 播放数据统计

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 听歌时长统计 | 统计累计听歌时长 | Room 数据库统计 |
| 高频播放歌曲/歌手 | 统计最常播放的歌曲和歌手 | 数据库聚合查询 |
| 月度/年度听歌报告 | 生成月度或年度听歌报告 | 数据统计 + 图表展示 |

### 3.2 进阶音效

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 重低音增强 | 增强低频音效 | Android Equalizer API |
| 环绕音效 | 模拟环绕立体声效果 | AudioEffect API |
| 人声增强 | 增强人声清晰度 | AudioEffect API |
| 自定义均衡器 | 用户可手动调节各频段 | Equalizer API |

### 3.3 本地音乐融合

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 扫描本地音频 | 读取手机本地存储的音频文件 | MediaStore API |
| 本地歌曲播放 | 播放本地音频文件 | ExoPlayer |

### 3.4 歌单导入导出

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 歌单分享 | 生成分享链接或文件 | JSON 序列化 |
| 批量导入歌曲 | 从文件或链接批量导入歌曲 | JSON 解析 |
| 歌单备份 | 备份歌单数据到云端或本地 | 文件存储 + 云端同步 |

### 3.5 音频格式兼容

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 无损格式支持 | 支持 FLAC、WAV 等无损音频格式 | ExoPlayer 原生支持 |

### 3.6 多用户管理

| 功能点 | 描述 | 技术实现 |
|--------|------|----------|
| 子账号管理 | 给家人朋友开子账号 | 用户系统 + 权限控制 |
| 共享曲库 | 子账号共享主账号曲库 | 数据库权限设计 |
| 独立数据 | 各自保留独立的歌单、收藏和播放记录 | 用户隔离 |
